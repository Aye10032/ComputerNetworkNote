# 3.3 差错控制

## 3.3.1 差错的差生

差错基本上都是由于噪声引起的。

### 1、噪声的种类

- 全局性
  - 由于线路本身电气特性所产生的**随机噪声**（热噪声），是信道固有的，随机存在的
  - 解决方法：提高信噪比
- 局部性
  - 外界特定的短暂原因所造成的**冲击噪声**，是产生差错的**主要原因**
  - 解决办法：通常利用编码技术来解决

### 2、差错的种类

- **位错**：0变1、1变0
- **帧错**
  - 丢失：【1】【3】
  - 重复：【1】【2】【2】【3】
  - 失序：【1】【3】【2】
